# CHANGELOG

## v0.1.0

### 2026-03-12

- Enabled native microphone recording with live visualization and automatic transcription.
- Implemented natural language search using vector embeddings and cosine similarity.
- Setup local embeddings using `fastembed-rs` with `nomic-embed-text` model.

### 2026-03-11

- Implemented a preflight splash screen to validate runtime dependencies and system readiness.
- Developed a first-run setup wizard for automatic model downloading and dependency management.
- Added support for audio import and transcription using bundled `ffmpeg` and `whisper-cli`.
- Integrated AI-powered document processing for automatic title generation, summarization, and tag extraction.
- Created a document library view with sorting, filtering, and management capabilities.
